Ejercicio de practica de node.js. Esta es una aplicacion que lee e imprime los archivos y directorios dentro de un directorio.

### Haremos uso de:
- Picocolors para colorear la consola
- Node File System
- Node Path
- Promesas
- Mapeo de un arreglo
- Variable global process
